## 🚀 部署步骤（3分钟完成）

### 第一步：上传到 GitHub

```bash
# 进入项目目录
cd "d:\MyDesktop\Food AI"

# 如果还没有 Git 仓库，先初始化
git init

# 添加所有文件
git add .

# 提交
git commit -m "准备部署到 Vercel"

# 关联 GitHub 仓库（第一次需要）
# 先在 GitHub 创建空仓库，然后：
git remote add origin https://github.com/你的用户名/Food-AI.git

# 推送到 GitHub
git push -u origin main
```

### 第二步：在 Vercel 部署

1. 访问 https://vercel.com
2. 点击 "Sign Up" 或 "Log In"（使用 GitHub 账号）
3. 点击 "Add New..." → "Project"
4. 找到你的 "Food-AI" 仓库，点击 "Import"
5. 配置页面：
   - **Framework Preset**: 选择 `Other`
   - **Build Command**: 留空
   - **Install Command**: `pip install -r requirements.txt`
6. 点击 "Environment Variables"
7. 添加两个环境变量：
   ```
   名称: ZHIPU_API_KEY
   值: 96c2f3dc023441738ea4ab27dc288dba.74edfBTCaWL5bhhj
   
   名称: ZHIPU_API_KEY_TEXT
   值: 022ac847f3384c28be276fcdf04c9892.lVw8PyzurjGIXhm2
   ```
8. 点击 "Deploy"
9. 等待 1-2 分钟

### 第三步：测试在线版本

访问 Vercel 提供的 URL，测试所有功能。

## 💡 提示

### Vercel 免费版限制
- ✅ 无限部署次数
- ✅ 100GB 带宽/月
- ✅ 自动 HTTPS
- ⚠️ Serverless 函数执行时间：10秒
- ⚠️ 不适合长时间运行的任务

### 如果遇到超时问题
AI 调用可能需要较长时间，如果 Vercel 免费版 10 秒超时：
- 方案 1：升级到 Vercel Pro（$20/月，60秒超时）
- 方案 2：使用 Render 部署（完全免费，无时间限制）

详细对比请查看：[部署指南.md](./部署指南.md)
