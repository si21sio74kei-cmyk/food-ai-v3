## 🏗️ 二、组件样式优化

### 2.1 卡片系统

**改进点:**
- 圆角从 `20px` → `24px` (var(--radius-lg))
- 内边距从 `24px` → `28px`
- 添加顶部渐变装饰条(悬停时显示)
- 阴影系统分级: sm / default / lg / hover
- 边框颜色调整为半透明棕色

**视觉效果:**
```css
.card {
    border-radius: var(--radius-lg);  /* 24px */
    padding: 28px;
    box-shadow: var(--shadow);
    border: 1px solid var(--card-border);
}

.card::before {
    /* 顶部渐变装饰条 */
    background: linear-gradient(90deg, var(--primary), var(--secondary), var(--tertiary));
}

.card:hover {
    transform: translateY(-6px);
    box-shadow: var(--shadow-hover);
}
```

### 2.2 按钮系统

**主要按钮 (.btn-primary):**
- 内边距: `16px 32px` (更宽敞)
- 圆角: `var(--radius)` (16px)
- 悬停效果: 上移3px + 增强阴影
- 点击波纹动画

**次要按钮 (.btn-secondary):**
- 白色背景 + 棕色边框
- 悬停时轻微上移
- 半透明棕色背景

**图标按钮 (.icon-btn):**
- 新增组件
- 48x48px 方形圆角按钮
- 适合工具栏和操作组

### 2.3 输入框优化

**改进:**
- 内边距增加: `16px 20px`
- 聚焦时上移2px (而非放大)
- Placeholder颜色统一为 `--text-muted`
- 边框颜色调整为半透明棕色

```css
.input-field:focus {
    border-color: var(--primary);
    transform: translateY(-2px);  /* 优雅的上移动画 */
    box-shadow: 0 0 0 4px rgba(166, 124, 82, 0.1);
}
```

### 2.4 单选按钮组

**视觉升级:**
- 选中状态使用伪元素渐变背景
- 文字层级通过 `z-index` 确保可读性
- 悬停时显示阴影
- 选中时缩放 + 上移组合动画

### 2.5 底部TabBar

**优化细节:**
- 高度: `70px` → `72px`
- 背景色与整体协调
- 图标大小: `24px` → `26px`
- 未激活图标灰度30% (视觉降噪)
- 激活时图标上移2px + 去灰度
- 指示器宽度: `40px` → `48px`
- 添加点击缩放反馈

```css
.tab-icon {
    filter: grayscale(30%);  /* 未激活时降低饱和度 */
}

.tab-item.active .tab-icon {
    transform: scale(1.25) translateY(-2px);
    filter: grayscale(0%);  /* 激活时恢复色彩 */
}
```

## 🎭 四、动画与交互

### 4.1 统一的缓动函数

所有动画统一使用:
```css
cubic-bezier(0.4, 0, 0.2, 1)
```

这是Material Design的标准缓动曲线,提供自然的加速减速效果。

### 4.2 关键动画

| 动画名称 | 应用场景 | 效果 |
|---------|---------|------|
| `fadeInUp` | 页面切换 | 淡入 + 上移 |
| `slideDown` | 顶部导航栏 | 从上方滑入 |
| `slideInLeft` | 结果展示区 | 从左侧滑入 |
| `pulse` | 统计卡片背景 | 呼吸脉动 |
| `shimmer` | 进度条 | 闪光扫过 |
| `badgePulse` | 徽章 | 轻微呼吸 |
| `spin` | 加载图标 | 旋转 |

### 4.3 悬停效果规范

**卡片:**
- 上移6px
- 阴影增强
- 顶部装饰条显现

**按钮:**
- 主要按钮: 上移3px + 阴影加深
- 次要按钮: 上移3px + 背景微变
- 图标按钮: 上移3px + 缩放1.05倍

**标签:**
- 上移2px
- 背景色加深
- 显示阴影

## 📊 七、性能优化

### 7.1 CSS变量系统

所有颜色、尺寸、阴影都使用CSS变量:
- 易于主题切换
- 减少重复代码
- 提升可维护性

### 7.2 硬件加速

以下属性触发GPU加速:
- `transform` (位移、缩放、旋转)
- `opacity` (透明度)
- `backdrop-filter` (毛玻璃效果)

避免使用会触发重排的屬性:
- ❌ `width`, `height` (动画中)
- ❌ `top`, `left` (改用transform)
- ✅ `transform: translate()` (替代top/left)

### 7.3 动画性能

- 使用 `will-change` 提示浏览器优化
- 限制同时运行的动画数量
- 复杂动画使用 `requestAnimationFrame`

## 🎯 八、设计原则总结

### 8.1 一致性原则

✅ 所有圆角使用统一变量 (`--radius-sm/md/lg/xl`)  
✅ 所有阴影使用分级系统 (`--shadow-sm/default/lg/hover`)  
✅ 所有动画使用相同缓动函数  
✅ 所有间距遵循8px网格系统  

### 8.2 层次原则

✅ 通过阴影大小区分卡片层级  
✅ 通过字体粗细区分信息重要性  
✅ 通过颜色饱和度区分激活状态  
✅ 通过透明度区分主次内容  

### 8.3 反馈原则

✅ 所有可点击元素都有悬停效果  
✅ 所有按钮都有点击反馈 (缩放/波纹)  
✅ 加载状态有明确指示 (旋转图标)  
✅ 操作结果有Toast提示  

### 8.4 情感化设计

✅ 使用emoji增强亲和力 (🍽️ 🌱 🤖)  
✅ 温暖配色传达友好感  
✅ 流畅动画带来愉悦感  
✅ 微交互细节体现用心
