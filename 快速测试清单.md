## 🔍 常见问题排查

### 问题1: 生成食谱时显示"网络错误"
**可能原因**:
1. 服务器未启动
2. API密钥无效
3. 网络连接问题

**解决方法**:
1. 确认`food_guardian_ai_2.py`正在运行
2. 检查控制台是否有API调用失败的详细错误
3. 检查网络连接

### 问题2: 刷新页面后出现不认识的摄入记录
**原因**: 数据文件中有历史测试数据

**解决方法**:
运行`清理数据并重新开始.bat`脚本

### 问题3: 拍照识菜失败
**可能原因**:
1. 智谱API密钥无效
2. 图片格式不支持
3. 网络问题

**解决方法**:
1. 检查API密钥配置
2. 使用JPG/PNG格式图片
3. 检查网络连接

### 问题4: 语音识别失败
**可能原因**:
1. 浏览器不支持MediaRecorder API
2. 麦克风权限被拒绝
3. 音频格式转换失败

**解决方法**:
1. 使用Chrome/Edge等现代浏览器
2. 允许麦克风权限
3. 检查控制台错误信息
