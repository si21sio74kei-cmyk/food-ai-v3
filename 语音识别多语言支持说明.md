# 🎤 语音识别多语言支持说明

## ✅ 功能概述

FoodGuardian AI 的语音识别功能现已**完全支持中英文自动识别**，并根据界面语言设置智能优化识别效果。

---

## 🌍 支持的语言

### 1. **中文（普通话）**
- ✅ 标准普通话
- ✅ 带口音的普通话（识别率可能略低）
- 示例："帮我生成番茄炒蛋的食谱"

### 2. **英文**
- ✅ 美式英语
- ✅ 英式英语
- 示例："Generate a recipe for tomato scrambled eggs"

### 3. **中英混合**
- ✅ 自动识别混合语言
- 示例："我想做tomato scrambled eggs"
- 示例："How to cook 番茄炒蛋"

---

## 🔧 工作原理

### 智能语言检测机制

```
用户说话 → 录音 → 发送到智谱GLM-ASR API
                          ↓
              根据界面语言设置决定识别模式：
                          ↓
        ┌─────────────────┼──────────────────┐
        ↓                 ↓                  ↓
   界面是中文       界面是英文          其他情况
   (zh-CN)         (en-US)            (auto)
        ↓                 ↓                  ↓
   优先识别中文     优先识别英文      自动检测语言
```

### 技术实现

#### 后端配置 (`food_guardian_ai_2.py`)

```python
# 从请求中获取语言设置（默认为auto自动检测）
language = request.form.get('language', 'auto')

data = {
    'model': 'glm-asr-2512',  # 智谱多语言ASR模型
    'stream': 'false'
}

# 如果指定了语言，添加到请求中
if language in ['zh', 'en']:
    data['language'] = language
```

#### 前端调用 (`templates/index.html`)

```javascript
// 获取当前语言设置
const currentLang = localStorage.getItem('fgai_language') || 'zh-CN';
const asrLanguage = currentLang === 'en-US' ? 'en' : 'auto';

// 传递语言参数
formData.append('language', asrLanguage);
```

## ⚙️ 配置选项

### 默认行为（推荐）
- **不修改任何代码**
- 系统会根据界面语言自动选择识别模式
- 中文界面 → 优先识别中文
- 英文界面 → 优先识别英文

### 强制指定语言（高级用法）

如果您希望**始终使用某种语言识别**，可以修改前端代码：

```javascript
// 强制使用中文识别（无论界面语言是什么）
const asrLanguage = 'zh';

// 强制使用英文识别
const asrLanguage = 'en';

// 自动检测（默认）
const asrLanguage = 'auto';
```

## 🎯 最佳实践

### 1. **清晰发音**
- 保持正常语速
- 避免背景噪音
- 距离麦克风10-30厘米

### 2. **选择合适的语言模式**
- 主要说中文 → 使用中文界面
- 主要说英文 → 使用英文界面
- 中英混合 → 任意界面都可以

### 3. **验证识别结果**
- 识别后会显示在文本框中
- 可以点击"✏️ 修改文字"进行修正
- 确认无误后再点击"✅ 确定"

## ❓ 常见问题

### Q1: 为什么有时候英文识别不准确？
**A**: 可能的原因：
- 口音较重
- 背景噪音干扰
- 语速过快
- 建议：尝试放慢语速，靠近麦克风

### Q2: 可以说方言吗？
**A**: 
- ✅ 粤语、四川话等常见方言有一定识别率
- ❌ 但准确率不如普通话
- 建议：尽量使用普通话

### Q3: 识别错误怎么办？
**A**: 
1. 点击"✏️ 修改文字"手动修正
2. 点击"❌ 取消"重新录音
3. 检查麦克风和网络连接

### Q4: 支持其他语言吗（如日语、韩语）？
**A**: 
- 目前主要支持**中文和英文**
- 其他语言可能能识别，但准确率较低
- 如需支持更多语言，需要更换ASR服务

## 🔬 技术细节

### 使用的API
- **服务商**: 智谱AI (Zhipu AI)
- **模型**: GLM-ASR-2512
- **特点**: 
  - 支持中英文自动检测
  - 高准确率（普通话 >95%）
  - 免费额度充足

### 音频格式
- **输入**: WebM (浏览器录制)
- **转换**: WAV (PCM 16-bit, 16kHz)
- **大小限制**: 最大25MB
- **时长限制**: 约60秒（取决于音质）

### 性能指标
- **识别速度**: 1-3秒（取决于网络）
- **准确率**: 
  - 普通话: 95%+
  - 英文: 90%+
  - 中英混合: 85%+
