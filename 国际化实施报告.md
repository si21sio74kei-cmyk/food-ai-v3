### 3. 核心功能特性

#### 🌐 语言切换机制
```javascript
// 用户点击国旗图标 → 触发 toggleLanguage()
// 1. 切换语言标识 (zh-CN ↔ en-US)
// 2. 保存到 localStorage (持久化)
// 3. 重新加载语言包
// 4. 应用翻译到所有 data-i18n 元素
// 5. 触发自定义事件通知其他模块
```

#### 💾 数据持久化
- 使用 `localStorage` 保存用户语言偏好
- 页面刷新后自动恢复上次选择的语言
- 首次访问默认使用中文

#### 🔄 动态翻译
- 支持嵌套键查找（如 "nav.home"）
- 支持参数替换（如 "Hello {name}"）
- 支持 placeholder、title 等属性翻译
- 缺失翻译时降级显示键名并警告

## 📝 使用说明

### 开发者
```javascript
// 在JavaScript中获取翻译
const text = window.i18n.t('recipe.generate_recipe');

// 带参数的翻译
const greeting = window.i18n.t('home.welcome', { name: 'John' });

// 手动切换语言
await window.i18n.switchLanguage('en-US');
```

### HTML标记
```html
<!-- 普通文本 -->
<div data-i18n="nav.home">首页</div>

<!-- 输入框placeholder -->
<input data-i18n-placeholder="recipe.ingredients_placeholder" />

<!-- 工具提示 -->
<button data-i18n-title="camera.title">📸</button>
```
