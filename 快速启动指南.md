## ⚡ 快速启动 (3步)

### 第1步: 安装依赖

```bash
pip install flask requests pillow pydub SpeechRecognition
```

**依赖说明:**
- `flask`: Web服务器框架
- `requests`: HTTP请求库
- `pillow`: 图像处理 (拍照识菜功能)
- `pydub`: 音频处理 (语音识别功能)
- `SpeechRecognition`: 语音识别库

## 🎯 核心功能使用指南

### 1️⃣ 智能食谱生成

**步骤:**
1. 点击底部导航栏的 "🍳 智能食谱"
2. 输入食材名称 (如: 番茄,鸡蛋,土豆)
3. 选择就餐人数和饭量系数
4. 点击 "✨ 生成食谱" 按钮
5. 等待AI生成 (约5-10秒)

**提示:**
- ✅ 可以输入多种食材,用逗号分隔
- ✅ 勾选 "使用冰箱食材" 会结合库存生成
- ✅ 饭量系数: 0.8(小胃) / 1.0(正常) / 1.2(大胃)

---

### 2️⃣ 营养摄入记录

**步骤:**
1. 点击 "📊 营养记录"
2. 输入今日各类食物摄入量 (克)
   - 蔬菜
   - 水果
   - 肉类
   - 蛋类
3. 点击 "💾 保存记录"
4. 查看营养评估结果

**功能:**
- ✅ 实时营养评估 (不足/达标/超标)
- ✅ 联合国营养标准对照
- ✅ 个性化改善建议
- ✅ 近7天历史记录

---

### 3️⃣ 冰箱库存管理

**添加食材:**
1. 点击 "🧊 冰箱管理"
2. 填写食材信息:
   - 名称 (如: 西红柿)
   - 数量 (如: 500)
   - 单位 (g/kg/个)
   - 保质期 (可选)
3. 点击 "➕ 添加"

**查看库存:**
- 列表显示所有食材
- 可删除过期或不需要的食材
- 生成食谱时会自动考虑库存

---

### 4️⃣ 拍照识菜

**步骤:**
1. 点击右上角 "📸" 图标
2. 上传食材照片
3. 等待AI识别 (约3-5秒)
4. 识别结果自动填入食材输入框

**支持格式:**
- JPG/JPEG
- PNG
- WEBP

**提示:**
- ✅ 照片越清晰,识别越准确
- ✅ 尽量拍摄单一食材
- ✅ 光线充足效果更好

---

### 5️⃣ 语音交互

**语音输入:**
1. 点击右上角 "🎤" 图标
2. 允许麦克风权限
3. 说出食材名称或问题
4. 自动转换为文字

**支持场景:**
- 语音输入食材
- 语音提问营养问题
- 语音查询食谱

**注意:**
- 首次使用需要安装语音识别库
- 需要稳定的网络连接
- 支持中文普通话

---

### 6️⃣ 采购清单生成

**步骤:**
1. 点击 "🛒 采购清单"
2. 输入想吃的菜品 (如: 土豆炖牛肉,番茄炒蛋)
3. 选择就餐人数
4. 勾选 "包含预算估算" (可选)
5. 点击 "📝 生成清单"

**输出内容:**
- ✅ 按超市区域分类的食材清单
- ✅ 精确用量 (克/个/毫升)
- ✅ 挑选新鲜食材的建议
- ✅ 储存方法和保质期提醒
- ✅ 替代方案建议
- ✅ 预算估算 (人民币)

---

### 7️⃣ AI对话助手

**步骤:**
1. 点击 "💬 AI对话"
2. 输入你的问题,例如:
   - "如何用豆腐做一道低脂菜?"
   - "儿童补钙吃什么好?"
   - "减肥期间怎么吃?"
3. 获取专业建议

**适用场景:**
- 营养咨询
- 烹饪技巧
- 饮食建议
- 健康知识

## 🔧 常见问题

### Q1: 启动失败,提示 "ModuleNotFoundError"

**解决方案:**
```bash
pip install flask requests pillow pydub SpeechRecognition
```

---

### Q2: 语音识别无法使用

**原因:** 缺少依赖库或麦克风权限

**解决方案:**
1. 安装依赖:
   ```bash
   pip install pydub SpeechRecognition
   ```
2. 安装 FFmpeg (pydub需要):
   - Windows: 下载 https://ffmpeg.org/download.html
   - macOS: `brew install ffmpeg`
   - Linux: `sudo apt-get install ffmpeg`
3. 浏览器中允许麦克风权限

---

### Q3: AI生成速度慢或失败

**原因:** 
- 网络连接问题
- API配额用完
- 服务器繁忙

**解决方案:**
1. 检查网络连接
2. 查看终端日志,确认错误信息
3. 稍后重试
4. 如需更换API Key,修改代码中的 `ZHIPU_API_KEY`

---

### Q4: 拍照识菜识别不准确

**优化建议:**
- ✅ 使用清晰、光线充足的照片
- ✅ 尽量拍摄单一食材
- ✅ 避免复杂背景
- ✅ 食材占据画面主要部分

---

### Q5: 数据保存在哪里?

**位置:** `fgai_local_data.json` (与应用同目录)

**包含数据:**
- 用户昵称
- 环保统计数据
- 人群标签
- 每日摄入记录
- 冰箱库存

**备份建议:**
定期复制此文件到安全位置

---

### Q6: 如何修改配色方案?

**方法:** 编辑 `templates/index.html` 文件

找到 `:root` 部分:
```css
:root {
    --primary: #A67C52;      /* 修改这里 */
    --secondary: #D4A574;    /* 修改这里 */
    /* ... */
}
```

修改后刷新浏览器即可生效。

---

### Q7: 如何在局域网内访问?

**步骤:**
1. 查看本机IP地址:
   - Windows: `ipconfig`
   - macOS/Linux: `ifconfig`
2. 在其他设备上访问: `http://你的IP:5000`

**示例:**
```
Running on http://192.168.50.20:5000
```
其他设备访问: `http://192.168.50.20:5000`
